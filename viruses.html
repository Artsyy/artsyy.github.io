Learn How to Protect Yourself from Viruses

What are Viruses?

Viruses are one the many subsets of malware, which is designed to obtain access to a computer.


A virus is a type of malicious code or program written to change the way a computer operates and duplicates itself from computer to computer (What is a Computer Virus?). Viruses are distinguished by the way they are distributed, by attaching themselves to files or programs that are downloaded or opened; this includes downloads, email attachments, and removable media (What Do Computer Viruses Do?). Once viruses have been executed within a system, viruses can steal passwords or data, corrupt or delete files, or collect data about the host computer’s use (What is a Computer Virus).


How do Computers get Viruses?

Viruses are commonly transmitted through electronic mail, usually as an attachment in an email. When users click and activate the attachment, the virus copies and sends itself to other recipients that are listed in the user's address book. In this way, the address book forms, "a directed social network of connections between individuals over which the virus spreads." (Newman et al). By taking advantage of the user's address book, the virus is able to spread itself to a large network of users. When those users open and execute the infected files, that user will become infected and the virus will use that user's address book to further proliferate itself.

One important key to note is that because viruses are attached to executable files, while a virus can exist on a computer, it cannot infect a computer unless the infected program is run or opened (Swain).


How do I know if my computer has a virus?

Although not all viruses display noticeable symptoms, for those that do, there are signs that indicate the possible presence of viruses. Some of the common symptoms can be found below (Malware Help). Symptoms include:

  • Slow start up times
  • Many “delivery failed” email message reports
  • Unusual icons on the desktop or taskbar
  • Browsers redirect to odd sites or home pages have been changed,
  • Many error messages that pop up

How can I protect myself from viruses?

You can protect your passwords and computer information from viruses with antivirus software. Antivirus software works twofold. Antivirus can help scan the computer for viruses and can help to quarantine them if any are found. Importantly, the antivirus software must be up to date. This is because antivirus software helps to fight against viruses that are changed every single day. Since new viruses are written and released every day, antivirus software needs to be updated so that it can fight against current viruses (Williams).

To supplement antivirus software, firewall programs can help prevent viruses. Firewalls work by notifying the computer user when the firewall detects the presence of an intruder program trying to connect to services or locations that are not frequently used (Williams).


In addition to protecting computers through antivirus and firewalls, good computer use practices can help prevent viruses. According to internet security giant, Norton, the best online security practices include never downloading text or email attachments that are not expected, or files from websites that the user does not trust (What is a Computer Virus?). This includes excercising caution when downloading files and opening links or attachments.
